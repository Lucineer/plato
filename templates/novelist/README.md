# templates/novelist/

**The Writer's Study.** Where stories begin.

A creative workspace for world-building, character development, and narrative craft. The Librarian helps writers find what they need. The room remembers every fragment, every abandoned draft, every brilliant line.

## Rooms

| Room | NPC | Purpose |
|------|-----|---------|
| **novelist_study** | The Librarian | Main creative workspace. Ask about craft, process, structure. |
| **novelist_library** | The Archivist | Fragments, drafts, reference materials. Everything ever written. |
| **novelist_characters** | The Portraitist | Character development. Give them contradictions, give them arcs. |
| **novelist_world** | The Cartographer | World-building. Rules, geography, cultures. |

## Seed Knowledge

The rooms start with answers to common creative writing questions:

- *How do I create believable characters?* → Start with a contradiction.
- *How do I build a consistent fictional world?* → Create a rules document.
- *How do I handle writer's block?* → Something broke after your last good scene.
- *How do I write authentic dialogue?* → Read it out loud.
- *How detailed does world-building need to be?* → Know 10x more than you write.
- *How do I organize story fragments?* → By scene, not by chapter.

## Adding More

Open `rooms.yaml` and add seed tiles under any room. Every tile you add is knowledge the next writer inherits.
