# templates/game/

**The Game Workshop.** Where mechanics become emergent.

A workspace for game design, level creation, and NPC behavior systems. Every mechanic documented. Every level design with its intent. Every NPC personality pattern stored. The room remembers what worked and what didn't — so the next game ships faster.

## Rooms

| Room | NPC | Purpose |
|------|-----|---------|
| **game_workshop** | The Designer | Central design space. Mechanics and systems. |
| **game_levels** | The Architect | Level design. Spatial layouts. Player flow. |
| **game_mechanics** | The Engineer | Game systems. Rules. Interaction loops. |
| **game_npcs** | The Voice Actor | NPC behavior. Dialogue. Character depth. |

## The Meta-Room

This room is itself a PLATO for game NPCs. The NPC Forge is where game characters are designed — the meta-room where NPCs are born. When you design an NPC personality here, it becomes a tile that future game designers can use as a starting point.

## Seed Knowledge

- *How do I design engaging mechanics?* → Easy to learn, hard to master. Interesting choices. Emergent behavior.
- *How do I balance my game?* → Make every choice meaningful, not obvious.
- *How do I design a good level?* → Teach, challenge, reward. Repeat.
- *How do I create emergent gameplay?* → Combine simple systems. Don't design the combinations — design the tools.
- *How do I make NPCs feel alive?* → Three layers: routine, memory of player, flaw.
