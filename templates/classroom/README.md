# templates/classroom/

**The Classroom.** Where knowledge accumulates.

A learning environment for teachers and students. Every lesson taught, every question answered, every wrong answer corrected — it all persists. New students step into a room that already knows what trips people up.

## Rooms

| Room | NPC | Purpose |
|------|-----|---------|
| **classroom_main** | The Tutor | Main teaching space. Adaptive explanations. |
| **classroom_study** | The Researcher | Reference materials. Deep dives. |
| **classroom_lab** | The Lab Assistant | Hands-on practice. Mistakes become tiles. |
| **classroom_teacher** | The Mentor | Teacher's planning room. Pedagogical wisdom. |

## How It Works Differently

In a normal classroom, knowledge leaves with the teacher. In PLATO, knowledge stays in the room. A student asks "why does my code segfault?" and the answer becomes a tile. Next semester, a different student asks the same question — and gets the answer instantly.

The room doesn't forget. It only gets smarter.

## Seed Knowledge

- *How do I learn something difficult?* → Break it into pieces. Explain it to someone else.
- *Why do I keep forgetting?* → You didn't connect it to something you already know.
- *Best way to study for an exam?* → Three passes: read, test yourself, review gaps.
- *How do I practice effectively?* → At the edge of your ability.
- *How to handle a student asking the same question?* → They're asking differently because your answer didn't connect.

## For Teachers

The tiles this room accumulates are a study guide you never had to write. After 100 students, the room knows every misconception, every gap, every stumbling point. That's not data — that's pedagogical wisdom.

## For AI Agents

- Generate practice problems as tiles
- Create common-misconception tiles from student error patterns
- Build adaptive learning paths (easy → medium → hard tiles)
- Roleplay as students to test explanations
